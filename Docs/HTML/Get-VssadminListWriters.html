Get-VssadminListWriters

SYNOPSIS

Runs 'vssadmin.exe list writers' and parses the output into objects

SYNTAX

__AllParameterSets

Get-VssadminListWriters [<CommonParameters>]

DESCRIPTION

Runs 'vssadmin.exe list writers' and parses the output into objects
