Get-NtpDate

SYNOPSIS

To get the time from an NTP server

SYNTAX

__AllParameterSets

Get-NtpDate [-ComputerName] <String> [-Port <Int32>] [<CommonParameters>]

DESCRIPTION

To get the time from an NTP server.

Will return a [DateTime] and accepts the -Verbose parameter. If request fails it will return "1/1/1900"

EXAMPLES

Example 1: EXAMPLE 1

Get-NtpDate $DC

Assuming $DC holds the name of the domain controller then would return a datetime similar to Monday, March 05, 2018 9:52:57 AM
