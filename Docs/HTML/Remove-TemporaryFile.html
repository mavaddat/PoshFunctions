Remove-TemporaryFile

SYNOPSIS

Cleans user profiles of temporary files and folders that are greater than 3 days

SYNTAX

Remove-TemporaryFile [[-Days] <Int32>] [-All] [-WhatIf] [-Confirm] [<CommonParameters>]

DESCRIPTION

Gets list of all local profiles, ignoring those belonging to 'NT Authority'. It then searches that profiles %TEMP% folder and removes top level files and folders older than $Days day old.
