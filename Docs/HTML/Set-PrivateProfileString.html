Set-PrivateProfileString

SYNOPSIS

To set data in an .ini file

SYNTAX

__AllParameterSets

Set-PrivateProfileString [[-File <String>]] [[-Section <String>]] [[-Key <String>]] [[-Value <String>]] [-CreateFile] [<CommonParameters>]

DESCRIPTION

To set data in an .ini file. .ini files are plain text that is categorized in section names. Within a section there are key, value pairs. An example .ini file content is as follows:

[Section1] Key1=Data1

EXAMPLES

Example 1: EXAMPLE 1

Set-PrivateProfileString -File test2.ini -Section Section3 -Key Key5 -value 'New Value' -CreateFile

Would put the following data into test2.ini [Section3] Key5=Newer Value
