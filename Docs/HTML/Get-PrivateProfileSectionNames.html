Get-PrivateProfileSectionNames

SYNOPSIS

To get the section names out of an .ini file

SYNTAX

__AllParameterSets

Get-PrivateProfileSectionNames [[-File <String>]] [<CommonParameters>]

DESCRIPTION

To get the section names out of an .ini file. .ini files are plain text that is categorized in section names. Within a section there are key, value pairs. An example .ini file content is as follows:

[Section1] Key1=Data1

EXAMPLES

Example 1: EXAMPLE 1

Assuming you have a file test2.ini that has the following content

[Section3] Key5=Newer Value [Section1] Key1=Some data

Get-PrivateProfileSectionNames -File test2.ini

Section3 Section1

PARAMETERS

-File

The path to the .ini file. Can be a relative path.
