Remove-QuotesFromCsv

SYNOPSIS

Removes quotes from a CSV data set. Can optionally set $Delimiter to another character.

SYNTAX

Remove-QuotesFromCsv [-csv] <String[]> [[-Delimiter] <Char>] [-WhatIf] [-Confirm] [<CommonParameters>]

DESCRIPTION

Removes quotes from a CSV data set. Can optionally set $Delimiter to another character.

EXAMPLES

EXAMPLE 1

'"f1","f2"' | Remove-QuotesFromCsv

Will return f1,f2

EXAMPLE 2

"`"f1`"`t`"f2`"" | Remove-QuotesFromCsv -Delimiter "`t" -verbose

Will return VERBOSE: Starting Remove-QuotesFromCsv f1 f2 VERBOSE: Ending Remove-QuotesFromCsv

EXAMPLE 3

$array | ConvertTo-Csv -NoTypeInformation | Remove-QuotesFromCsv

Will return blah,blah,blah
