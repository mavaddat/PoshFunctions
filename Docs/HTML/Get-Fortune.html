Get-Fortune

SYNOPSIS

Display a short quote

SYNTAX

__AllParameterSets

Get-Fortune [[-Path <String>]] [[-Delimiter <String>]] [-Speak] [<CommonParameters>]

DESCRIPTION

Display a short quote from a file which defaults to: 'c:\scripts\wisdom.txt') but can be changed with parameter -Path.
