Get-VssWriter

SYNOPSIS

Runs 'vssadmin.exe list writers' and parses the output into objects

SYNTAX

__AllParameterSets

Get-VssWriter [<CommonParameters>]

DESCRIPTION

Runs 'vssadmin.exe list writers' and parses the output into objects

Get-VssWriter | Where-Object { $_.VssWriter -eq 'Registry Writer' }
