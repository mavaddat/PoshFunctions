Format-WrapText

SYNOPSIS

Wraps text at a particular column width

SYNTAX

Width (Default)

Format-WrapText [-Text] <String[]> [-Width <Int32>] [<CommonParameters>]

Screen

Format-WrapText [-Text] <String[]> [-Screen] [<CommonParameters>]

DESCRIPTION

Wraps text at a particular column width (Default=80). Function aliased to 'WrapText'.

EXAMPLES

Example 1: EXAMPLE 1

Format-WrapText -Text "word1 word2 word3 word4 word5" -Width 12

Would return word1 word2 word3 word4 word5
