Get-List

SYNOPSIS

Returns an array given an indeterminate number of command line arguments.

SYNTAX

__AllParameterSets

Get-List [<CommonParameters>]

DESCRIPTION

Returns an array given an indeterminate number of command line arguments.

EXAMPLES

Example 1: EXAMPLE 1

$Colors = Get-List blue red green

Creates an array of 3 string elements: 'blue', 'red', 'green'
