Invoke-Beep

SYNOPSIS

Uses the Beep function of the .Net [Console] class

SYNTAX

Default (Default)

Invoke-Beep [-Duration <Int32>] [-Frequency <Int32>] [<CommonParameters>]

Birthday

Invoke-Beep [-Birthday] [<CommonParameters>]

DESCRIPTION

Uses the Beep function of the .Net [Console] class. Alias as 'Beep'. Has the ability to play a number of different pre-defined set of tones.

EXAMPLES

Example 1: EXAMPLE 1

Invoke-Beep

Example 2: EXAMPLE 2

Invoke-Beep -Frequency 400 -Duration 500

Example 3: EXAMPLE 3

Invoke-Beep -Impossible
