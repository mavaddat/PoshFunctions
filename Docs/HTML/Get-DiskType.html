Get-DiskType

SYNOPSIS

A quick function to determine what type of local disk a computer has

SYNTAX

__AllParameterSets

Get-DiskType [[-ComputerName <String[]>]] [<CommonParameters>]

DESCRIPTION

A quick function to determine what type of local disk a computer has

EXAMPLES

Example 1: EXAMPLE 1

Get-DiskType

Query the local machine's disk to determine its bus and media types

Example 2: EXAMPLE 2

Get-DiskType -ComputerName server01

Example output ComputerName DeviceId BusType MediaType

SERVER01 0 SAS HDD SERVER01 1 SAS HDD

PARAMETERS

-ComputerName

A string (or string array) of computers that you want checked. Defaults to $env:COMPUTERNAME
