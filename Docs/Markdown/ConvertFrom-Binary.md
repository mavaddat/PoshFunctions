# ConvertFrom-Binary

## SYNOPSIS
Convert an string or string array from binary to an integer

## SYNTAX

```
ConvertFrom-Binary [-Binary] <String[]> [-IncludeInput] [<CommonParameters>]
```

## DESCRIPTION
Convert an string or string array from binary to an integer

## EXAMPLES

### EXAMPLE 1
```
ConvertFrom-Binary -Binary 100001
```

33

### EXAMPLE 2
```
ConvertFrom-Binary -Binary 1001 -IncludeInput
```

Binary Number
------ ------
1001        9
