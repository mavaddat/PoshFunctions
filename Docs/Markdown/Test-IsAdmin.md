# Test-IsAdmin

## SYNOPSIS
Determines if you are running elevated (as Administrator)

## SYNTAX

```
Test-IsAdmin [<CommonParameters>]
```

## DESCRIPTION
Determines if you are running elevated (as Administrator)

## EXAMPLES

### EXAMPLE 1
```
Test-IsAdmin
```

Would return the following if the prompt was elevated.
$True

### EXAMPLE 2
```
Test-IsAdmin -Verbose
```

Would return the following if the prompt was elevated.
VERBOSE: You have Administrator rights.
True
