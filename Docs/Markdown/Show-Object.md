# Show-Object

## SYNOPSIS

Takes an object and displays a new window containing the object, and you can drill down on its properties.

## SYNTAX

### __AllParameterSets

```
Show-Object [[-InputObject <Object>]] [<CommonParameters>]
```

## DESCRIPTION

Takes an object and displays a new window containing the object, and you can drill down on its properties.


## EXAMPLES

### Example 1: EXAMPLE 1

```
Show-Object (Get-Date)
```
