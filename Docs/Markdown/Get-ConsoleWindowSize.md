# Get-ConsoleWindowSize

## SYNOPSIS
Gets the current the window size and buffer size of the current console window

## SYNTAX

```
Get-ConsoleWindowSize [<CommonParameters>]
```

## DESCRIPTION
Gets the current the window size and buffer size of the current console window

## EXAMPLES

### EXAMPLE 1
```
Get-ConsoleWindowSize
```

BufferWidth BufferHeight WindowWidth WindowHeight
----------- ------------ ----------- ------------
        196        32766         196           21
