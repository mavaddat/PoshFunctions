# Get-RandomMacAddress

## SYNOPSIS
Gets a random sequence of 12 hexadecimal digits

## SYNTAX

```
Get-RandomMacAddress [<CommonParameters>]
```

## DESCRIPTION
Gets a random sequence of 12 hexadecimal digits

## EXAMPLES

### EXAMPLE 1
```
Get-RandomMacAddress
```

# Would return a result like
6eee1e05cb29
