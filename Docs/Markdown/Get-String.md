# Get-String

## SYNOPSIS

Returns a string given an indeterminate number of command line arguments.

## SYNTAX

### __AllParameterSets

```
Get-String [<CommonParameters>]
```

## DESCRIPTION

Returns a string given an indeterminate number of command line arguments.


## EXAMPLES

### Example 1: EXAMPLE 1

```
$String = Get-String Hello   World 1
```

Creates a variable containing the string 'Hello World 1'
