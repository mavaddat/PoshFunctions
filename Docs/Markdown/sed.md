# sed

## SYNOPSIS
A simple text filter to replace strings

## SYNTAX

```
sed [-Before] <String> [-After] <String> [<CommonParameters>]
```

## DESCRIPTION
A simple text filter to replace strings

## EXAMPLES

### EXAMPLE 1
```
'Hello There' | sed 'Hello' 'Goodbye'
```

Would return
Goodbye There
